Data Type Ranges
===

Typical value range in C++.
Skipping `unsigned *`.

[Data Type Ranges](https://docs.microsoft.com/en-us/cpp/cpp/data-type-ranges?view=msvc-160)

## 1. int-like
### (1) short (2bit)
-32,768 ～ 32,767

≒ `± 3 * 10^4`

### (2) int (4bit)
-2,147,483,648 ～ 2,147,483,647

≒ `± 2 * 10^9`

### (3) long long (8bit)
-9,223,372,036,854,775,808 ～ 9,223,372,036,854,775,807

≒ `± 9 * 10^18`
